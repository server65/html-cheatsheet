HTML CheatSheet
  1. HTML Boilerplate:This is the basic HTML boilerplate code 
    <!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <link rel="stylesheet" href="">
        <title>Document</title>
    </head>
    <body>
    </body>
</html>
  2. Div Section 
    <div>Block element</div>
  3. Heading 
    <h1>Page title (Largest)</h1>
<h2>Subheading (Larger) </h2>
<h3>Tertiary heading (Large)</h3>
<h4>Quaternary heading (Smaller)</h4>
  4. Paragraph 
    <p style="text-align: center;">text</p>
  5. Image 
    <img src="/demo.jpg" alt="description" height="48" width="100" />
  6. Bold Text 
    <strong>Bold text</strong>
  7. Italic Text 
    <em>Italic text</em>
  8. Underlined Text 
    <span style="text-decoration: underline;">Underlined text</span>
  9. Video 
    <video width="200" height="150" controls>
<source src="vid.mp4" type="video/mp4">
<source src="vid.ogg" type="video/ogg">
No video support.
</video>
  10. Audio 
    <audio controls>
<source src="sound.ogg" type="audio/ogg">
<source src="sound.mp3" type="audio/mpeg">
No audio support.
</audio>
